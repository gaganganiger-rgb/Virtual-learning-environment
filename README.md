# 🎓 Virtual Learning Environment

A full-stack web-based platform for online courses and interactive learning, designed to make education accessible and engaging.

---

## 🚀 Features

- 📚 Browse and enroll in online courses
- 🎥 Interactive learning modules
- 👤 Student & instructor dashboards
- 📊 Progress tracking for learners
- 🔐 User authentication & profile management
- 💬 Discussion and Q&A sections

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, HTML, CSS |
| Backend | Python, Node.js |
| Database | SQL (MySQL) |
| Version Control | Git & GitHub |
| OS | Windows 11 |

## ⚙️ Getting Started

### Prerequisites
- Python 3.x
- Node.js
- MySQL
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/gaganganiger-rgb/Virtual-learning-environment.git

# Navigate to project folder
cd Virtual-learning-environment

# Install backend dependencies
pip install -r requirements.txt

# Install frontend dependencies
cd frontend
npm install

# Start the app
npm start
```

### Database Setup

```sql
-- Create the database
CREATE DATABASE virtual_learning;
USE virtual_learning;

-- Run the schema file
SOURCE database/schema.sql;
```
